This box has a shadow

We added the shadow to this box, by using the CSS property "box-shadow". It does about what you'de expect a thing called "box-shadow" to do. A lot of people think that a subtle shadow makes your images and containers look really professional.

Using box-shadow is easy. The syntax goes:

box-shadow {10px 10px 10px #000000;}

The first number is how far to the right or left the shadow is offset, the second is how far up or down, the third number is how hazy the shadow is, and the last thing is color.
